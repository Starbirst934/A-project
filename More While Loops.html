What is Break?

You can use the break command to stop a while loop from iterating before it ends naturally. Here's the syntax.

xxxxxxxxxx
 
while (condition1):
  if(condition2):
      break

You can also use break in for loops.

xxxxxxxxxx
 
# If condition2 becomes true while the for loop is running, break stops the loop.
for i in range(number):
    if(condition2):
        break

While / Else and For / Else

Just like with the if statement, you can also use else statements with while and for loops, too. The else clause allows you to execute commands when the loop finishes.

While-Else

When the while loop condition is True this loop will continue to run. When the value becomes False the else code will execute.

xxxxxxxxxx
 
# for "while" loop
while (condition):
  # commands
else:
    # commands

For-Else

When i is still in the range provided by the range() function this loop will act normally. When i leaves the range, the else code will execute.

xxxxxxxxxx
 
# for "for" loop
for i in range(number):
  # commands
else:
​
  # commands

At first glance, the else clause might seem pointless. But it's useful because the code in else statements only get executed if the loop finishes naturally. In other words, any commands in your else won't execute if a loop is stopped by a break command. else and break give you more fine-grained control of exactly how your program executes.

Continue

The continue instruction ends the current iteration of the nearest enclosing loop. Unlike the break instruction, it does not end the loop, only the current iteration.

xxxxxxxxxx

Notice how there is no 3 squared is 9, or 6 squared is 36, or 9 squared is 81 in the output. This is because the continue command will move onto the next iteration of the loop. In this case, continue is called when n is a multiple of 3. When n = 3, the computer will print skipping 3 instead of 3 squared is 9. Remove the continue command to see the difference in the output.
